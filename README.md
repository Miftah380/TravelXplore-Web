# TravelXplore 🌍

TravelXplore is a responsive web application designed to provide real-time global travel intelligence. Built as a final project submission, it integrates the REST Countries Fetch API to dynamically retrieve and display geographic data for over 190 countries.

## Features ✨
- **Dynamic Search:** Asynchronous search functionality to find countries globally without page reloads.
- **Interactive Maps:** Direct integration with Google Maps Search API for accurate destination exploration.
- **Corporate UI:** Modern, premium interface featuring glassmorphism elements, custom animations, and responsive grids.
- **Error Handling:** Dynamic UI states for loading processes and "Not Found" errors.

## Tech Stack 💻
- **Structure:** HTML5
- **Styling:** TailwindCSS (via CDN) & Custom CSS
- **Logic:** Vanilla JavaScript (ES6+)
- **Data Provider:** [REST Countries API](https://restcountries.com/)

## Project Structure 📁
- `index.html` - Premium Landing Page
- `destinations.html` - Core Search & API Integration Feature
- `about.html` - Corporate Identity & Infrastructure Info
- `/css/style.css` - Custom styling and animations
- `/js/app.js` - Global scripts (if externalized)
